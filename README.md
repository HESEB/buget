# Insect Collector Game Prototype

이 프로젝트는 곤충 채집 게임의 간단한 웹 프로토타입입니다. 모바일 세로 화면을 기준으로 설계되었으며, 20종의 곤충을 더미 이미지로 표시합니다. 나중에 `assets/insects/` 폴더의 PNG 파일을 사용자가 만든 이미지로 교체하여 커스터마이징할 수 있습니다. GPS 정보와 가방, 설정, 가위바위보 포획 시스템, 도감 페이지 등 포켓몬고 스타일의 요소를 간략히 구현하였습니다.

## 사용 방법

1. `insect_game` 폴더의 `index.html` 파일을 웹 브라우저로 열면 게임이 시작됩니다.
2. 브라우저에서 GPS 권한을 허용하면 화면 상단에 현재 위도·경도 좌표가 표시되며, 맵 중앙에 붉은 원으로 플레이어 마커가 나타납니다. 이동하면 마커도 위치가 변하여 대략적인 이동을 확인할 수 있습니다. geolocation을 지원하지 않거나 권한을 거부하면 “GPS 사용 불가”로 표시됩니다.
3. 맵 영역에 무작위로 나타난 곤충 아이콘을 눌러 포획을 시도합니다. 클릭하면 가위·바위·보 중 하나를 선택하는 포획 모달이 열립니다. 기본 성공률은 30%이며 먹이 아이템을 사용하면 한 번 동안 성공 확률이 상승합니다. 장갑을 착용한 수만큼 포획 시도가 가능하며, 포획에 성공하면 도감에 등록됩니다. 모든 데이터는 브라우저 `localStorage`에 저장됩니다.
4. 하단의 **도감** 버튼을 클릭하면 한 곤충씩 넘겨보는 책 형태의 도감이 나타납니다. 각 페이지에는 큰 이미지, 설명, 포획 마릿수가 표시되며 '이전/다음' 버튼으로 이동할 수 있습니다. **닫기** 버튼으로 모달을 닫습니다.
5. **가방** 버튼을 클릭하면 아이템 목록이 나타납니다. 포획 장갑은 사용 시 장갑 사용 가능 수를 5 증가시키며, 먹이는 다음 포획 성공률을 올려줍니다. 루어는 30초 동안 곤충 스폰 수를 늘리고, 플래시는 5분 동안 야간에도 포획이 가능하도록 합니다. 각 아이템은 사용 후 개수가 감소합니다.
6. 맵에는 일정 확률로 **아이템 박스**가 출현합니다. 박스를 클릭하면 임의의 아이템을 하나 획득하여 가방에 추가됩니다. 또한 하루에 한 번, 아이템과 장갑이 자동으로 소량 회복됩니다. 자동 회복 여부는 당일 최초 실행 시 표시됩니다.
6. **설정** 버튼에서 야간 모드를 켜고 끌 수 있습니다. 야간 모드가 켜지면 맵이 어두워지며 실제 시간에 따라 밤낮 여부와 플래시 사용 여부에 따라 포획이 제한될 수 있습니다. 사운드 토글은 후속 버전에서 확장할 수 있습니다.
7. 화면을 리사이즈하면 곤충 아이콘과 플레이어 마커가 새 위치에 다시 배치됩니다.
8. 곤충 출현 수는 기본 8마리로 제한되며 야간/주간과 계절에 따라 각 곤충의 출현 확률이 달라집니다. 예를 들어 반딧불이는 야간과 여름에 더 많이 나타나고, 매미는 여름에 많이 등장합니다. 루어를 사용하면 일시적으로 출현 수가 두 배로 증가합니다.
8. `assets/insects/` 폴더의 PNG 이미지를 원하는 곤충 이미지로 대체하면 게임 내 아이콘이 변경됩니다. 파일명은 기존과 동일하게 유지해야 합니다.

## 구조

- `index.html` – 기본 페이지 구조를 정의합니다.
- `style.css` – 레이아웃과 스타일을 정의합니다.
- `script.js` – 곤충 데이터를 로드하고 맵에 랜덤 배치하는 로직을 포함합니다.
- `assets/insects/` – 각 곤충의 더미 이미지 20개를 저장하는 폴더입니다. 현재는 회색 배경과 번호만 표시되며 교체용입니다.

## 참고

이 버전은 브라우저의 geolocation API를 통해 GPS 좌표를 표시하며, 플레이어의 이동을 간략히 반영한 마커를 지도 위에 표시합니다. 포획한 곤충 데이터를 도감에 저장하고 가방과 설정 기능, 야간 모드, 아이템 효과 등을 포함합니다. 실제 지도를 불러오는 기능과 AR 요소, 서버와의 통신은 포함되어 있지 않으므로 상용 서비스에는 지도 SDK 연동, 서버 기반 데이터 관리, UI/UX 개선 등이 추가로 필요합니다.
